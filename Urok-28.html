Анонімні класи

Анонімні класи – це класи, які не мають імені.
Зазвичай такі класи корисні, якщо необхідно один раз створити об’єкт такого класу. І більше цей клас не використовуватиметься. Наприклад: 
         
            $person = new class {};
Для визначення змінної аонімного класу, як у випадку зі звичайним класом, застосовується оператор new , після якого йде ключове слово class та потім – тіло класу у фігурних дужках.
Анонімні класи, як і звичайні класи можуть визначати властивості та методи. Наприклад: 
         
            $person = new class {
               public $name;
                
               function sayHello() 
               {
                   echo "Hello!
";
               }
           };
           $person->sayHello();
           $person->name = "Sam";
           echo "Name: " . $person->name . "
";
         
      
         
            Hello!
            Name: Sam
Також анонімні класи можуть визначати конструктори: 
         
            $person = new class( "Bob" ) 
            {
               public $name;
               
               function __construct( $name )
               {
                  $this->name = $name;
               }
               
               function sayHello() 
               {
                  echo "Hello!
";
               }
            };
            echo "Hello, " . $person->name . "
"; // Hello, Bob
При цьому виклик конструктора йде відразу після ключового слова class до фігурної дужки, що відкриває: new class("Bob") {. Подібним чином можна визначати властивості відразу в конструкторі, скоротивши цим визначення класу: 
         
            $person = new class( "Bob", 34 ) 
            {
               function __construct( public $name, public $age ) 
               {
                  $this->name = $name;
               }
               
               function displayInfo() 
               {
                  echo "Name: $this->name; Age: $this->age
";
               }
            };
            $person->displayInfo();
         
      
         
            Name: Bob; Age: 34
