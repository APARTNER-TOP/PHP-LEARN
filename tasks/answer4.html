2) PHP Створити функцію з іменем calc , яка буде приймати три параметри , третій параметр буде не обовязковим. У функції викликати перевірку на наявність третього параметру. Отримати результат усіх введених параметрів.
function calc($a, $b, $c = null) { // Check if the third parameter is provided if(isset($c)) { // If the third parameter is provided, // calculate the result of all three parameters $result = $a + $b + $c; } else { // If the third parameter is not provided, // calculate the result of the first two parameters $result = $a + $b; } // Return the result return $result; }
