Завдання 1: Реєстрація користувача: Створити форму реєстрації, яка містить поля для введення ім'я користувача, електронної пошти, номеру телефона та пароля. При отриманні форми, зберегти дані нового користувача в базі даних. Захешувати пароль, використовуючи, наприклад, функцію password_hash() в PHP, перед збереженням його в базі даних. Авторизація користувача: Створити форму авторизації, яка містить поля для введення електронної пошти та пароля. При отриманні форми, перевірити, чи існує користувач з введеним електронним адресом в базі даних. Порівняти хешований пароль збережений в базі даних з введеним паролем, використовуючи функцію password_verify() в PHP. Встановити сесію або використовувати токен для позначення авторизованого користувача. Оновлення даних користувача: Дозволити авторизованому користувачу оновлювати свої дані, такі як ім'я, електронна пошта, телефон або пароль. Створити форму оновлення даних, яка містить поля для введення нових значень. При отриманні форми, оновити відповідні дані користувача в базі даних. Видалення користувача: Дозволити авторизованому користувачу видалити свій аккаунт. Створити форму для підтвердження видалення аккаунта. При отриманні форми, видалити дані користувача з бази даних. Чат: Створити форму для відправки повідомлення в чаті, яка містить поле для введення тексту повідомлення. При отриманні форми, зберегти повідомлення в базі даних, разом з інформацією про автора, дату та час. Вивести всі повідомлення, збережені в базі даних, відсортовані за датою та часом. Оновлення повідомлення: Дозволити користувачу вибрати конкретне повідомлення для оновлення. При отриманні запиту на оновлення, оновити відповідний запис повідомлення в базі даних. Видалення повідомлення: Дозволити користувачу вибрати конкретне повідомлення для видалення. При отриманні запиту на видалення, видалити відповідний запис повідомлення з бази даних.
