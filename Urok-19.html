У PHP є велика кількість математичних функцій, які дозволяють виконувати різноманітні математичні операції. У цьому уроці ми розглянемо такі функції як пошук мінімуму та максимуму, генерація випадкових чисел та округлення. Пошук мінімуму та максимуму Для пошуку мінімуму та максимуму в PHP є дві функції: min() та max(). Вони дозволяють знайти мінімальне та максимальне значення з масиву чисел. Приклад використання функції min(): 
         
            $numbers = array(4, 2, 8, 5, 1);
            $min = min($numbers);
            echo "Мінімальне число: " . $min;
Приклад використання функції max(): 
         
            $numbers = array(4, 2, 8, 5, 1);
            $max = max($numbers);
            echo "Максимальне число: " . $max;
Генерація випадкових чисел У PHP є функція rand(), яка дозволяє генерувати випадкові числа. Функція приймає два параметри: мінімальне та максимальне значення. Приклад використання функції rand(): 
         
            $randomNumber = rand(1, 100);
            echo "Випадкове число: " . $randomNumber;
Округлення У PHP є декілька функцій для округлення чисел: round(), ceil() та floor(). Функція round() дозволяє округлити число до заданої кількості знаків після коми. Функції ceil() та floor() дозволяють округлити число до ближчого цілого вище або нижче. Приклад використання функції round(): 
         
            $number = 3.14159265359;
            $roundedNumber = round($number, 2);
            echo "Округлене число: " . $roundedNumber;
Якщо число має бути округлене вгору до найближчого цілого числа, ви можете використати функцію ceil(). Приклад використання функції ceil(): 
         
            $number = 3.14159265359;
            $ceiledNumber = ceil($number);
            echo "Округлене число: " . $ceiledNumber;
Функція floor() використовується для округлення чисел до меншого цілого числа. Синтаксис функції: 
         
            floor(float $number) : float
Приклад використання функції floor(): 
         
            $number = 123.456789;
            $flooredNumber = floor($number); // $flooredNumber = 123
Функція number_format() Функція number_format() використовується для форматування чисел з роздільниками тисяч і знаками після коми. Синтаксис функції: 
         
            number_format(float $number, int $decimals = 0, string $decPoint = ".", string $thousandsSep = ",") : string
         
      
         
            $number = 1234567.891234;
            $formatted_number = number_format($number, 2, '.', ',');
            echo $formatted_number;
У цьому прикладі: Перший параметр - число, яке необхідно сформатувати. Другий параметр - кількість знаків після коми. Третій параметр - символ, який буде використаний як десятковий роздільник. Четвертий параметр - символ, який буде використаний як роздільник для розрядів. Ця функція може бути корисною для форматування чисел для відображення користувачам або для експорту даних у формати, що вимагають певного формату чисел. Результат виводу: 
         
            1,234,567.89
Ще одна корисна функція - abs() для отримання модуля числа. 
         
            $number = -123;
            $abs = abs($number);
            echo $abs; // 123
Також існують функції для розрахунку тригонометричних функцій, логарифмів, експоненти та інших математичних операцій. Наприклад, для обчислення синуса кута в радіанах можна використовувати функцію sin(): 
         
            $angle = pi() / 2; // 90 градусів в радіанах
            $sin = sin($angle); // 1
Або для обчислення натурального логарифму числа - функцію log(): 
         
            $number = 10;
            $log = log($number); // 2.302585092994
У цьому уроці ми розглянули декілька основних математичних функцій мови PHP. Ці функції дозволяють здійснювати різноманітні математичні операції над числами. Знання цих функцій є необхідним для розробки складних алгоритмів та обробки даних в програмуванні.
